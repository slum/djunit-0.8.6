djunit 0.8.6 for eclipse 4.5.2
====

# 動作条件
* 下記のいずれか。
 * "Eclipse IDE for Java Developers 4.5.2" with "Eclipse 2.0 Style Plugin Support" plugin
 * "pleiades-e4.5-java_20160229.zip(eclise 4.5.2)"

# インストール方法
* ZIPダウンロードして解凍後、「ecilpse/dropins」に配置してください。
* pleiades の場合は一度「eclipse.exe -clean.cmd」で起動してください。

```
dropins
└─djunit-0.8.6
    └─eclipse
        └─plugins
            └─jp.co.dgic.eclipse.jdt.djunit_0.8.6
```
